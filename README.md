# Портфолио Екатерины Шулятниковой

Портфолио ведущей премиальных мероприятий из Пермского края.

## Технологии

- **HTML5** - структура страницы
- **CSS3** - стилизация (встроенные стили)
- **JavaScript (Vanilla)** - интерактивность и анимации
- **Tailwind CSS** - утилитарные CSS-классы (через CDN)
- **Font Awesome** - иконки (через CDN)
- **Vite** - инструмент сборки и dev-сервер

## Установка и запуск

### Требования

- Node.js версии 18 или выше
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск dev-сервера

```bash
npm run dev
```

После запуска dev-сервер будет доступен по адресу: `http://localhost:3000`

Сервер автоматически откроет браузер и будет отслеживать изменения в файлах с горячей перезагрузкой (HMR).

### Сборка для production

```bash
npm run build
```

Собранные файлы будут находиться в директории `dist/`.

### Просмотр production-сборки

```bash
npm run preview
```

## Особенности

- Адаптивный дизайн для всех типов устройств
- Плавные анимации и переходы
- Элегантный минималистичный дизайн
- Оптимизированная производительность
- SEO-дружественная структура

## Разработка

Проект использует Vite для быстрой разработки с горячей перезагрузкой модулей. Все изменения в `index.html` автоматически отображаются в браузере без необходимости перезагрузки страницы.
