# Руководство по адаптивному дизайну Portfolio Theme

Это руководство описывает структуру адаптивного дизайна темы Portfolio Theme и систему breakpoints.

## Структура Breakpoints (Mobile First)

Тема использует подход Mobile First, что означает, что базовые стили предназначены для мобильных устройств, а медиа-запросы добавляют стили для больших экранов.

### Breakpoint 1: Extra Small Devices (320px - 479px)

**Целевые устройства:**
- iPhone SE (320px)
- Старые Android телефоны (360px)
- Маленькие мобильные устройства

**Особенности:**
- Бургер меню с фиксированным боковым меню
- Минимальные размеры шрифтов для читаемости
- Одна колонка для всех сеток
- Увеличенные touch-targets (минимум 44px)
- Оптимизированные отступы для маленьких экранов

**Примеры размеров:**
- Header Logo: 16px
- Hero заголовок: clamp(32px, 8vw, 40px)
- Hero подзаголовок: clamp(14px, 3.5vw, 16px)
- Секции padding: 48-64px вертикально, 20px горизонтально

### Breakpoint 2: Small Devices (480px - 767px)

**Целевые устройства:**
- iPhone 12/13/14 (390px, 393px, 430px)
- Samsung Galaxy S21 (360px)
- Большие телефоны, маленькие планшеты

**Особенности:**
- Бургер меню с оптимизированным боковым меню
- Две колонки для некоторых сеток (Stats, Recognition)
- Улучшенные размеры шрифтов
- Адаптивные сетки для портфолио

**Примеры размеров:**
- Header Logo: clamp(18px, 4.5vw, 20px)
- Hero заголовок: clamp(48px, 12vw, 56px)
- Hero подзаголовок: clamp(16px, 4vw, 18px)
- Секции padding: 64-80px вертикально, 24px горизонтально

### Breakpoint 2.5: Medium Small Devices (768px - 899px)

**Целевые устройства:**

**Особенности:**

- Две колонки для некоторых сеток
- Улучшенные размеры для планшетов
- Оптимизированные отступы

**Примеры размеров:**
- Header Logo: clamp(20px, 2.5vw, 22px)
- Hero заголовок: clamp(64px, 8vw, 72px)
- Hero подзаголовок: 18px

## Специальные Медиа-запросы

### Landscape Mobile (Мобильные устройства в альбоме)

**Медиа-запрос:** `@media screen and (max-width: 767px) and (orientation: landscape)`

**Особенности:**
- Уменьшенная высота Hero секции (min-height: 500px)
- Уменьшенные размеры шрифтов
- Скрыт scroll индикатор
- Оптимизированные отступы секций (48px вместо 64px)
- Уменьшенная высота изображений

### Touch Device Optimizations

**Медиа-запрос:** `@media (hover: none) and (pointer: coarse)`

**Особенности:**
- Все интерактивные элементы: min-height 44px, min-width 44px
- Увеличенные отступы между элементами
- Автоматический показ hover эффектов через Intersection Observer
- Оптимизация скролла для touch устройств
- Предотвращение zoom на iOS (font-size: 16px для полей ввода)

### High DPI Displays (Retina)

**Медиа-запрос:** `@media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi)`

**Особенности:**
- Улучшение качества изображений
- Антиалиасинг шрифтов
- Оптимизация рендеринга для Retina дисплеев

### Reduced Motion (Доступность)

**Медиа-запрос:** `@media (prefers-reduced-motion: reduce)`

**Особенности:**
- Отключение всех анимаций
- Отключение transitions
- Отключение parallax эффектов
- Мгновенное отображение элементов

## Touch Optimizations

Все интерактивные элементы на touch устройствах должны иметь:

- **Минимальная высота:** 44px
- **Минимальная ширина:** 44px (для кнопок и ссылок)
- **Padding:** минимум 12px по горизонтали
- **Gap между элементами:** минимум 16px

**Элементы, которые требуют touch оптимизации:**
- Кнопки (hero-cta, portfolio-cta, testimonials-cta, contact-cta)
- Ссылки в навигации
- Элементы форм (input, select, textarea, button)
- Карточки (service-card, testimonial-card, contact-card)
- Элементы портфолио
- Модальные окна

## Рекомендации по использованию

1. **Используйте clamp()** для плавного масштабирования размеров шрифтов
2. **Используйте min-height: 44px** для всех интерактивных элементов на touch устройствах
3. **Используйте единую систему отступов** для консистентности
4. **Тестируйте на реальных устройствах** перед деплоем
5. **Проверяйте горизонтальный скролл** на всех breakpoints
6. **Учитывайте landscape режим** для мобильных устройств
